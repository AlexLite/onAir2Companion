# v0.1.0

Первая публичная версия интеграции Soft-Lab FDOnAir с Bitfocus Companion через SLMessageServer.

## В релизе

- Bridge-сервер `onAir2Companion` (Windows)
- Runtime-пакет `release/server`
- Модуль Companion `release/companion/onair-bridge-0.1.0`

## Важно

- Базовая конфигурация рассчитана на `localhost`
- Тестирование маршрутизации выполнялось с `http`-транспортом
- Для других транспортов может потребоваться дополнительная доработка
